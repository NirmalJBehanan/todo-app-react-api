# 📝 Todo App using React useReducer

A simple and responsive Todo Application built with React and the useReducer Hook for state management.

## 🚀 Live Demo

🌐 Live Site: https://todoapp-react-api.netlify.app/

## 📂 GitHub Repository

🔗https://github.com/NirmalJBehanan/todo-app-react-api

## ✨ Features

- Add new todos
- Edit existing todos
- Delete todos
- Update todo status
  - Completed
  - Not Completed
- Filter todos by status
- State management using React useReducer
- Responsive UI

## 🛠️ Technologies Used

- React.js
- JavaScript (ES6+)
- CSS3
- useReducer Hook

The application uses React's `useReducer` hook to manage complex state updates through actions and a reducer function. :contentReference[oaicite:0]{index=0}

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/NirmalJBehanan/todo-app-react-api
```

Navigate to the project folder:

```bash
cd to-do-app-useReducer
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## 🎯 Reducer Actions

### ADD_TODO
Adds a new todo item.

### EDIT_TODO
Updates an existing todo.

### DELETE_TODO
Removes a todo from the list.

### UPDATE_STATUS
Changes the status of a todo.
